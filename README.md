
# FIRECLI
基于Google Python Fire库，设计的一套命令行模式下具有MVC理念的微型框架


FIRECLI把那些烦人的定位参数，可选参数等等全部进行了封装，你只要专注业务逻辑部分。

不需要花很大的力气在熟悉argparse等库存的用法上面。节省的大量的精力和时间。
